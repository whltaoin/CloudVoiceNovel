# 前后端技术栈详细说明

## 前端技术栈

### 框架与库
- **React** - 前端UI开发框架
- **TypeScript** - 类型安全的JavaScript超集
- **React Native** - 跨平台移动应用开发
- **Capacitor** - 跨平台应用容器，支持iOS/Android原生功能

### UI组件库
- **React Native Elements** 或 **Ant Design Mobile** - UI组件库
- **React Navigation** - 应用导航解决方案

### 状态管理
- **Redux** 或 **MobX** - 全局状态管理
- **React Query** - 数据获取与缓存

### 音频处理
- **react-native-track-player** - 音频播放管理

## 后端技术栈

### 语言与框架
- **Node.js** - JavaScript运行时
- **Express.js** 或 **Koa.js** - Web框架
- **TypeScript** - 类型安全开发

### 数据库
- **MongoDB** - NoSQL数据库（存储用户信息、小说元数据等）
- **MySQL** 或 **PostgreSQL** - 关系型数据库（存储结构化数据）
- **Redis** - 缓存系统

### AI服务
- **语音合成API** - 文本转语音服务
- **推荐算法** - 个性化内容推荐

### 认证与安全
- **JWT** - 无状态认证
- **OAuth2.0** - 第三方授权
- **HTTPS** - 加密传输

### 部署与容器化
- **Docker** - 容器化技术
- **Kubernetes** - 容器编排

## 中间件与工具

### API文档
- **Swagger** - API文档生成与测试

### 日志与监控
- **Winston** - 日志管理
- **Prometheus** - 监控系统

### 其他工具
- **ESLint/Prettier** - 代码规范与格式化
- **Jest** - 单元测试框架
